# Phase 5: Data Analytics & Insights - Day 21

## 1. Flight Delay Classification

- **On-time**: Arrived within 15 minutes of scheduled time (Aviation standard).
- **Delayed**: Arrived > 15 minutes late.
- **Canceled**: Flight status is 'Canceled'.

## 2. Calculated Fields

- Using `AGE()` or simple subtraction for duration.
- Handling multi-timezone calculations with `AT TIME ZONE`.
